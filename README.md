# 🧠 NeuroNotes AI

NeuroNotes AI is an AI-powered lecture assistant built using Streamlit.

It helps students by:

✅ Converting lecture audio into text (Transcription)  
✅ Generating smart summaries  
✅ Creating quiz questions automatically  
✅ Showing performance analysis  
✅ Exporting report as PDF  

---

## 🚀 Features

• Login & Create Account System  
• Upload lecture (.mp3 / .wav / .mp4)  
• AI Transcription using Whisper  
• Auto Summary Generation  
• Smart Quiz Generator  
• PDF Report Download  

---

## 🛠 Tech Stack

• Python  
• Streamlit  
• OpenAI Whisper  
• NLTK  
• ReportLab  

---

## ▶️ How to Run

1. Clone repository:

git clone https://github.com/rushikeshk-tech/NeuroNotes-AI.git

2. Install dependencies:

pip install -r requirements.txt

3. Run project:

streamlit run app.py
